How to Install a Custom CSS Theme

Follow these steps to add a custom theme to your VRCX application:

  1. Download the Theme:

    Download the themeâ€™s zip file from the VRCX Themes Page.

  2. Extract the Files:

    Extract the downloaded zip file and locate the custom.css file.

  3. Place the CSS File:

    Navigate to your VRCX folder and place the custom.css file there.

    1. Press Win + R on your keyboard to open the Run dialog.
    2. Type %appdata%\VRCX and press Enter.
    3. Place the custom.css file in the VRCX folder.
    4. Ensure the file is named custom.css for it to apply correctly.

    Alternatively:

    1. Go to VRCX > Settings > Advanced.
    2. Click the AppData (VRCX) button to open the folder.
    3. Place the custom.css file in the opened folder.
    4. Ensure the file is named custom.css for it to apply correctly.
  4. Reload the Theme:

    Press ALT + SHIFT + R to reload the theme and see your changes.
